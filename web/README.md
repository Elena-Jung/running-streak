# 자랑 카드 페이지 (`/자랑`)

`index.html` 은 러닝 스트릭 봇의 **마일스톤 자랑 카드 생성기**입니다. 외부 요청이 전혀 없는
**자기완결 정적 파일**(HTML + inline CSS/JS, 순수 `<canvas>` 렌더)이라, 봇/DB 컨테이너와 무관하게
아무 정적 호스트로 서빙하면 됩니다.

## 동작
- 봇의 `/자랑` 커맨드가 통계(현재 연속일·마일스톤 티어·총거리·총시간·평균페이스·이름)를
  **URL 프래그먼트(`#`)** 에 담아 링크를 줍니다.
- 이 페이지가 `#` 값을 읽어 카드를 그립니다. 사용자는 **배경 사진을 고르고**(업로드 아님, 브라우저
  안에서만 처리) **이미지로 저장**합니다.

## 개인정보
- 서버에 저장되는 것: **없음**. 사진은 브라우저 밖으로 나가지 않고, 통계는 `#` 프래그먼트라
  **서버로 전송조차 되지 않습니다**(접근 로그에도 안 남음).
- 페이지는 `noindex` 로 검색 색인을 막습니다.

## 서빙 방법 (예)
- 리버스 프록시가 있으면 이 폴더를 한 사이트의 문서 루트로 지정.
  - Caddy: `run.example.org { root * /path/to/repo/web; file_server }`
  - nginx: `root /path/to/repo/web;`
- 로컬 테스트: `python3 -m http.server -d . 8080` → `http://localhost:8080`.

서빙한 공개 주소를 봇 `.env` 의 `BRAG_BASE_URL` 에 넣으십시오(예 `https://run.example.org`).
비워 두면 `/자랑` 은 링크 없이 통계를 텍스트로만 보여줍니다.
